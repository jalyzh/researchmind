# 示例研究报告

> 这是一个示例，展示 ResearchMind Agent 生成的报告样式。

---

## 研究概述

本报告展示了 ResearchMind AI Agent 自动生成研究报告的能力。

### 工作流程

1. **规划** - Agent 将研究主题分解为子问题
2. **搜索** - Agent 联网搜索每个子问题
3. **综合** - Agent 综合所有信息生成最终报告

---

## 示例输出预览

当你运行 ResearchMind 后，会得到类似以下的 Markdown 报告：

```markdown
# 小米MiMo大模型的技术特点与应用场景

> 生成时间: 2026-04-28 12:00:00
> 模型: gpt-4o

## 目录

1. [研究概述](#1-研究概述)
2. [子问题研究](#2-子问题研究)
3. [综合结论](#3-综合结论)
4. [参考来源](#4-参考来源)
```

---

## 快速开始

```bash
# 1. 设置 API Key
set LLM_API_KEY=your_api_key_here

# 2. 运行研究
researchmind "小米MiMo大模型的技术特点"
```
